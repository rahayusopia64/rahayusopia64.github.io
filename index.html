Biodata Rahayu

Hallo, nama saya rahayu sopia Ramadhani. Biasanya orang orang memanggil saya ayu atau ucha. Umur saya 17 tahun. Saya lahir di bogor pada 28 november 2002. Sagitarius adalah zodiak saya. Berat badan saya 47 kg dan tinggi badan saya 168 cm. Saya tinggal di bogor semenjak saya lahir. Tepatnya di kampung lebak jaya rt01 rw06 no.42 desa sukaresmi kecamatan tamansari dengan kode pos 16610. Saya sangat suka masakan padang padahal semua keluarga saya adalah asli sunda. Saya suka makanan padang karena hampir semua makanannya pedas, dan saya sangat suka pedas. Saya bersekolah di SMK-SMAK Bogor, dimana asalah sekolah saya adalah SMP Negeri 4 Bogor

Saya sangat suka menonton. Menonton apa saja asalkan menghibur. Contohnya menonton film. Hampir seluruh genre film saya suka namun hanya satu genre film yang saya tidak suka, yaitu film horror. Karena saya sangat penakut. Selain menonton film saya juga menonton konser musik. Hal hal tersebut yang sangat menghibur saya
